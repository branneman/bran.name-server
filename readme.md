# Playground: Critical CSS served via HTTP/2 from Node.js

```
git clone git@github.com:branneman/playground-nodejs-http2-criticalcss.git
cd playground-nodejs-http2-criticalcss
npm install
npm start
```

Open browser: https://localhost:8443/
